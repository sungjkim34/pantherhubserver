Welcome to the PantherHub server.

/getAllStudents [GET]

Gets a list of all student objects

/getStudent/:studentId [GET]

Gets the student object by id

  • studentId - int

/addStudents [POST]

Adds a student

  • firstName - string
  • lastName - string
  • dob - string [YYYY/MM/DD format]
  • major - string
  • startDate - string [YYYY/MM/DD format]
